# QR Code Generator 🔥

A Python CLI tool to generate QR codes with limited colors and optional logo.

## Features
- Generate QR from text or URL
- Custom file name
- Custom colors
- Optional logo support
- Error handling
- Handle Data of any size

## Installation
pip install -r requirements.txt

## Example
Input: https://www.youtube.com/@Formula1  
Output: QR code image generated

## Tech Stack
- Python
- qrcode
- Pillow

## Run
python main.py
