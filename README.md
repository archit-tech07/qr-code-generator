# 🚀 QR Code Generator

A customizable QR Code Generator built using Python.
This tool allows users to generate QR codes from text or URLs with custom colors and optional logo embedding.

---

## ✨ Features

* Generate QR codes from text or URLs
* Custom foreground and background colors
* Optional logo embedding
* Automatic file saving and opening
* Input validation

---

## 📦 Installation

pip install -r requirements.txt

---

## ▶️ Usage

python main.py

---

## 🧪 Example

Input:
https://www.youtube.com/@Formula1

Output:
QR code image generated and saved locally.
